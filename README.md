# CNN-Transformer-SHAP-MHAW

This repository presents a convolutional neural network with a transformer-based model hypertunded by Bayesian optimization for time series forecasting. The optimized model is available [here](https://github.com/SFStefenon/CNN-Transformer-SHAP-MHAW/blob/main/proposed_model.py). 

To compute the experiments and validate the model's performance, the natural inflow of the Tucurui hydroelectric power plant was considered (available [here](https://github.com/SFStefenon/CNN-Transformer-SHAP-MHAW/blob/main/tucurui.csv)).

To have these hyperparameters set up, hyperparameter tuning was computed using Bayesian optimization as presented [here](https://github.com/SFStefenon/CNN-Transformer-SHAP-MHAW/blob/main/Model_eval/hypertuning.py). To prove that the proposed model is stable, a statistical analysis was computed, as presented [here](https://github.com/SFStefenon/CNN-Transformer-SHAP-MHAW/blob/main/Model_eval/stats.py).
